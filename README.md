# CRUD APP

This is the most generic way of writing CRUD application in MEAN Stack

## Installation
Clone the repo and run frontend and backend server seperately

### For Frontend
```bash
cd frontend
npm install
ng build --watch
```
### For Backend
```bash
cd backend
npm install
node server.js
```
